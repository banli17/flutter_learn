# flutter_template

Flutter 项目开发模版。

## 简介

## v1.0 

**功能**

- 闪屏页
- 底部导航
- 页面切换
- 支持 webview, 使用 `webview_flutter`。

## 说明

### 修改闪屏页

在 `pubspec.yaml` 配置：

```
flutter_native_splash:
  image: assets/splash.png
  color: "42a5f5"
  fill: true
```
